1.HTML - HTML (Hypertext Markup Language) is the standard markup language for creating web pages and web applications. It provides the structure and content of a webpage through a system of tags and attributes, enabling the display of text, images, links, and multimedia on the internet.

2.CSS - CSS (Cascading Style Sheets) is a stylesheet language used for describing the presentation and visual design of web pages, allowing you to control the layout, colors, forms, and other stylistic aspects of HTML elements.

3.JAVASCRIPT - JavaScript is a versatile programming language for web development, enabling interactive and dynamic behaviour in web applications, and providing client-side scripting capabilities.
